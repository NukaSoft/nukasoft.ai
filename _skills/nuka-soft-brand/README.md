# Nuka-Soft Brand -- Creative Direction

Nuka-Soft Brand is a Claude Code skill that gives your AI assistant deep knowledge of the NukaSoft brand identity -- including visual guidelines, tone of voice, character backstory for Rita Rivera, and creative specifications for all branded content.

## What It Does

- **Brand bible enforcement**: Color palettes, typography, logo usage, layout compositions
- **Rita Rivera character guide**: Full backstory, visual eras, voice rules, and writing guidelines
- **Tone of voice system**: Dual-layer messaging (1950s optimism surface, survival subtext)
- **Asset management**: Reference image catalog for logos, posters, and character art
- **Creative extensions**: Flavor variants, merchandise concepts, environmental design specs
- **Persona: Lando** -- Creative Director, named after Lando Calrissian

## Installation

1. Copy the `nuka-soft-brand/` folder to `~/.claude/skills/nuka-soft-brand/`
2. Place brand reference images in the `assets/` subdirectory
3. Claude will automatically use these guidelines when creating NukaSoft content

## Usage

Claude will reference this skill when you mention NukaSoft, Rita Rivera, the rocket bottle, or any retrofuturistic branding work. Examples:

- "Write ad copy for NukaSoft"
- "What are the brand colors?"
- "Write something as Rita"
- "Design a new poster layout"
- "What's the rocket bottle look like?"
