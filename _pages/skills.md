## Installation

```bash
# Copy any skill to your Claude Code skills directory
cp -r skills/bishop ~/.claude/skills/bishop

# Or clone the whole repo and symlink
git clone https://github.com/NukaSoft/nukasoft.ai.git
ln -sf nukasoft.ai/_skills/bishop ~/.claude/skills/bishop
```

---

*Each skill is a standalone Claude Code skill file. Copy it to `~/.claude/skills/` and you're running.*
